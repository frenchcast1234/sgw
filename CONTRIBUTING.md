# Contributing to Secret Games Wiki

Thank you for your interest in contributing!

## How to contribute

1. Fork this repository
2. Create a new branch: `git checkout -b my-contribution`
3. Make your changes
4. Commit your changes: `git commit -m "Add: description of my change"`
5. Push to your branch: `git push origin my-contribution`
6. Open a Pull Request on this repository

## What you can contribute

- Add missing Google games
- Fix typos or incorrect information
- Improve existing pages

## Rules

- Keep information accurate and sourced
- Write in English
- Be respectful in Pull Requests
